## This sample console app provides .net core concepts (including basic oops)

# Create dotnet console app using
- $ mkdir consoleapp
- $ cd consoleapp
- $ dotnet new console
- $ dotnet build
- $ dotnet run

## - Program.cs internally calls
    - OOPSPrac.cs - Basic OOPS concepts like to sum two numbers
        a) Simple Class
        b) Abstract Class
        c) Interface
        d) Sealed Class
        e) Encapsulation - using properties
        f) Polymorphism 
    
    - IntermediateConcepts.cs
        a) Delegate - method reference and anonymous
        b) Thread - Basic Thread start and parameterized thread
        c) Generic - int and double addition
